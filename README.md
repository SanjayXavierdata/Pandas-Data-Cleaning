# Pandas Data Cleaning Project

## Project Overview

This project demonstrates a complete data cleaning workflow using Python and Pandas. The objective was to transform a raw customer contact dataset into a clean, consistent, and analysis-ready dataset.

## Dataset

The dataset contains customer contact information, including names, phone numbers, addresses, and contact preferences.

### Data Quality Issues Identified

- Duplicate records
- Inconsistent phone number formats
- Unwanted characters in text fields
- Missing values
- Inconsistent categorical values
- Customers marked as "Do Not Contact"

## Data Cleaning Tasks Performed

- Removed duplicate records
- Standardized phone number formats
- Cleaned text fields using string operations
- Split address information into separate columns
- Handled missing values
- Standardized categorical values (Yes/No)
- Removed customers who requested not to be contacted
- Prepared the dataset for further analysis
